# Formal Convo Simulator
A simulator that generates scenarios based on certain topics where one can submit their response to the scenario and get feedback on their response in a formal setting.
The simulator is currently based on LLM-generated feedback and responses. A community of professionals and users is planned for future releases, as human feedback can help users build confidence in communication.


## Workflow
1. User types or searches their topic of conversation.
2. User selects a scenario and submits a response.
3. User gets LLM-generated feedback and a chained reply to that response.


## Features
- User can practice scenario-based conversations in formal settings ranging from applying for a job to handling a client or talking with their manager.
- Users earn points for completing exercises, boosting motivation.
- Users receive follow-up conversations generated from their responses.


## Planned Features
- User can post their response and get constructive feedback from other professionals.
